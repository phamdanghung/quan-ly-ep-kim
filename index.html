An toàn dữ liệu xưởng

Dữ liệu được lưu trữ trên trình duyệt này. Hãy tải file sao lưu để lưu trữ ngoài.

📥 Tải file sao lưu (Export)

Xuất toàn bộ dữ liệu bài ép và thợ ép hiện tại ra tệp `.json` để cất giữ an toàn.

📤 Khôi phục dữ liệu (Import)

Đọc tệp `.json` sao lưu để khôi phục lại trạng thái cũ của xưởng. *Dữ liệu hiện tại trên trình duyệt sẽ bị thay thế hoàn toàn.*
